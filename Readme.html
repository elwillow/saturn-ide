INSTALLATION

None. Just unzip and run the Saturn executable. If you're planning on using it to read/write to the Archive you might need to give it permissions (dependent on OS and KSP location).

PACKER

The packer allows you to create "deploy" configurations to do several optimizations to your code.

Destination. Can either be clipboard, file or the Archive. When it comes to the archive you can specify to save with the same name as your program currently has or a new one. It won't check for existing files, it will just overwrite anything with the same name.

Remove comments. Will remove any comments you have in your code. The empty lines that might result from this will be kept (they don't occupy any space in kOS).

Remove leading spaces. If you use spaces to tabulate your code you can check this options and all those spaces will be removed. Trailing spaces are always removed.

Run deep var collision scan. The packer will go through your code looking for programs run by yours, and programs run by programs run by yours, and pro... and check if there are any variables with the same name. This differs from the real time checking in the IDE by the dept of the scan, the real time scan checks variables only in programs run by yours.

Remove lines after tag. This functionality allows you to mark portions of your code that you want to keep in a debug version of the program, but you don't want in your final code.

For example, if you add the rule "DEBUG-LEVEL-1" (case insensitive):

print "A bunch of code here".
//DEBUG-LEVEL-1
print "Current vessel altitude: " + ship:altitude.
log time + ": " + ship:altitude to MyLog.

print "and more here".

Your code will be stripped from the line "//DEBUG-LEVEL-1" to the first empty line, thus leaving:

print "A bunch of code here".

print "and more here".


SHORTCUTS NOT SHOWN ON MENUS

Some handy shortcuts that are not visible in any menu and you might not be aware of:

Ctrl-Backspace: Delete to beginning of word.
Ctr-T: Delete to end of word.
Ctrl-Y: Delete whole line.
Ctrl-Shift-Y: Delete to EOL.
Ctrl-Shift-B: Move to matching bracket.
Alt: Momentary toggle column selection mode to use with Shift + Cursor/PgUp/PgDown/Home/End.
Ctrl-Shift-<number>: Toggle bookmark <number>, from 0 to 9.
Ctrl-<number>: Go to bookmark <number>.
Ctrl-Space: Code completion.
